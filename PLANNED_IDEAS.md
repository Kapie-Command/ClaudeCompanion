# ClaudeCompanion — Planned Enhancement Ideas

1. **Favorites / Pinning** — Let users pin frequently-used templates to the top for quick access
2. **Search & Filter** — A search bar across templates by name, tags, or content
3. **Import / Export** — Share template collections as JSON files between machines or teammates
4. **Keyboard Shortcuts** — e.g. Ctrl+N for new template, Ctrl+1/2/3 to quick-copy pinned templates
5. **Template Categories / Folders** — Organize templates into groups (debugging, code review, architecture, etc.)
6. **Markdown Preview** — Live preview of the interpolated prompt before copying
7. **Snippet Variables** — Reusable variable presets (e.g. "React project" fills in language=TypeScript, framework=React at once)
